# **App Name**: HACK 'A' WAR

## Core Features:

- Dynamic Hero Presentation: Showcase the project title 'HACK 'A' WAR' and team details ('Heapify', 'GRC', 'Regulatory Intelligence Agent') with large glitch, neon glow, and smooth fade-in animations on load.
- Animated Problem & Solution Storytelling: Visually present the problem statement ('Fragmented regulatory data', 'Manual compliance overload', 'Lack of real-time intelligence') and the application's solution workflow through animated cards, typing animations, and a vertical timeline with motion transitions.
- Interactive Feature Showcase: Highlight core product features such as 'Real-time regulatory tracking', 'AI-powered risk scoring', and 'Automated compliance alerts' using uniquely styled, animated glassmorphism cards with hover effects and typing animations.
- AI Regulatory Query Demonstrator: Provide an interactive tool where users can input a simulated regulatory query, and the application generates a mock AI-driven insight or solution demonstration, illustrating the core value of 'HACK 'A' WAR'.
- Technical Stack Visualization: Display the underlying technology stack (AWS, OpenAI, React, Firebase, Tailwind CSS) via an infinite horizontal scrolling logo marquee and an animated flowchart illustrating the system architecture.
- Responsive & Performance-Optimized UI: Ensure a fully responsive experience across mobile, tablet, and desktop, coupled with high-performance animations and smooth transitions using Framer Motion and an advanced scrolling library like Lenis.
- Team & Impact Presentation: Introduce the team members and summarize the product's impact ('Reduce compliance cost', 'Improve decision-making') using animated, glassmorphism cards and a strong closing statement.

## Style Guidelines:

- A dark, futuristic color palette leveraging 'Black' (#000000) for a deep background, 'Primary' green (#76B900) for branding and key interactive elements, and 'Accent Neon' green (#75ff00) for striking glow effects and dynamic highlights. 'White' (#FFFFFF) provides crisp text contrast, while 'Dark Grey' (#5e5c63) and 'Light Grey' (#aaaaaa) offer subtle hierarchy in typography and secondary UI components.
- All text will primarily use the 'Space Grotesk' font (sans-serif), chosen for its modern, computerized, and tech-aligned aesthetic, suitable for both headlines and concise body text to maintain a cohesive futuristic feel.
- Geometric and minimalist line icons for tech stack representation and UI elements, frequently enhanced with neon glow effects. Subtle grain and noise overlays will be applied to UI elements to enrich the futuristic texture and depth.
- A two-page single-page application experience with full-viewport sections for a cinematic journey. Content is structured with smooth parallax scrolling background layers for depth, vertical storytelling timelines, dynamic horizontally scrolling logo marquees, and interactive animated flowcharts to guide the user through complex information.
- Extensive use of Framer Motion for high-performance and cinematic interactions, including large-scale glitch effects, vibrant neon glows, precise typing and typewriter animations, and seamless fade-ins. Interactive elements feature subtle hover effects (scale, glow), while a custom cursor glow enhances user interaction. All animations will be tied to smooth scrolling (e.g., Lenis) and utilize custom easing functions for a fluid, premium feel.
